Activity Stack
- 기본 : 후입선출(LIFO, Last In, First Out)
    FIFO : 선입선출 (First In, First Out)
- stack 될 수 있으면 건들지 않는 게 좋다
- 분명한 의도를 가지고 적용해야 한다


Stack 관리
- Manifest 이용
    - launchMode
        - standard(LIFO)
        - singleTop
        - singleTask
        - singleInstance

- Intent Flag 이용
    - FLAG_ACTIVITY_NEW_TASK (singleTask)
    - FLAG_ACTIVITY_SINGLE_TOP (singleTop)
    - FLAG_ACTIVITY_CLEAR_TOP
